# solana-nft-minter

**1.Setup**

```bash
yarn
```

**2.Run script (CLI)**

```bash
yarn start
```

**2.Run script (One Click Minting)**

```bash
yarn mint <One Click Minting config file path>
```

Example:

```bash
yarn mint ./one-click-minting-config/example.json
```
